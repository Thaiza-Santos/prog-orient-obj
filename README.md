# Estudo Dirigido 1 - Programação Orientada a Objetos
### 1º Bimestre - Repositório: programacao-oo

## 📝 Descrição
O objetivo deste Estudo Dirigido é a prática dos conceitos iniciais de **Programação Orientada a Objetos (POO)** em Dart, focando na criação de classes, definição de atributos, uso de construtores e implementação de métodos. O projeto abrange desde validações simples até o conceito de **composição de objetos**.
